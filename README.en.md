# 智能字符串

#### Description
智能字符串，用于快捷进行文字处理，当前主要功能是提供缩略语功能，可以添加常用的缩略语，对于任何的电脑工作者均有巨大帮助，可以快捷的打印出任意文字。
例如：
dh] -> 13128812409 （此为本人的微信，如果有需求建议以及赞助可加本人微信，备注：智能字符串）
sfz] -> 你的身份证号
无论在中英文状态下，均可以打印出来。

软件基本功能完善，其余有趣功能陆续开发中。
